# Automated Bus Scheduling and Route Management System

A web-based system developed to automate bus scheduling, route management,
crew assignment, and ticket booking for public transport.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- React Context API
- JSON & localStorage

## Key Features
- Admin dashboard for managing buses, routes, schedules, and crew
- Automated scheduling with conflict prevention
- Crew login with assignment and check-in/check-out
- Public user ticket booking with seat selection
- Emergency handling and real-time updates

## How to Run the Project
1. Install dependencies  
   npm install
2. Start development server  
   npm run dev
3. Open browser at  
   http://localhost:3000
